* 项目说明
本项目是一个简历 LaTeX 模板, fork 自 [[https://github.com/Boese0601/Curriculum-Vitae-Latex]], 进行了简化, 移除了硬编码的路径. 

* 目录结构
#+begin_src text
.
├── .gitignore
├── .gitmodules
├── LICENSE
├── fonts (as gitsubmodule)
└── resume
    ├── .latexmkrc
    ├── figures
    │   ├── avatar.jpg
    │   └── your_logo.eps
    ├── main.tex
    └── packages
        └── resume.sty
#+end_src

* 环境依赖
- TeX 发行版: 建议 TeX Live 或 MiKTeX.
- 编译引擎: XeLaTeX (由 =resume/.latexmkrc= 默认指定).
- 构建工具: latexmk.
- 字体依赖: 文档中使用 =TeX Gyre Termes=, =FontAwesome=, =Adobe CJK= 字体族(如 =Adobe Song Std L=, =Adobe Heiti Std R=, =Adobe Kaiti Std R=, =Adobe Fangsong Std R=). 有关字体文件请见 =fonts= 指向的 branch, 或者从原仓库中下载获得. 

* 子模块
`fonts` 目录由 Git 子模块管理, 子模块信息定义在 `.gitmodules`.

初始化与更新子模块:
#+begin_src bash
git submodule update --init --recursive
#+end_src

* 快速开始
1. 进入简历目录.
#+begin_src bash
cd resume
#+end_src

2. 清理中间文件(可选).
#+begin_src bash
latexmk -c main.tex
#+end_src

3. 使用 latexmk 编译.
#+begin_src bash
latexmk -pdf main.tex
#+end_src
