* 项目说明
本项目是一个简历 LaTeX 模板, fork 自 [[https://github.com/Boese0601/Curriculum-Vitae-Latex]], 进行了简化, 移除了硬编码的路径. 

* 目录结构
#+begin_src text
.
├── .gitignore
├── .gitmodules
├── LICENSE
├── fonts (as gitsubmodule)
└── resume
    ├── .latexmkrc
    ├── figures
    │   ├── avatar.jpg
    │   └── your_logo.eps
    ├── main.tex
    └── packages
        └── resume.sty
#+end_src

* 环境依赖
- TeX 发行版: 建议 TeX Live 或 MiKTeX.
- 编译引擎: XeLaTeX (由 [[resume/.latexmkrc][.latexmkrc]] 指定).
- 构建工具: =latexmk=.
- 字体依赖: 文档中使用 =TeX Gyre Termes=, =FontAwesome=, =Adobe CJK= 字体族(如 =Adobe Song Std L=, =Adobe Heiti Std R=, =Adobe Kaiti Std R=, =Adobe Fangsong Std R=). 有关字体文件请见 [[https://github.com/ren-lingyu/resume-template/tree/fonts][fonts]] 分支, 或者从[[https://github.com/Boese0601/Curriculum-Vitae-Latex][原仓库]]中下载获得. 

* 子模块
字体文件置于 [[https://github.com/ren-lingyu/resume-template/tree/fonts][fonts]] 分支, 作为 =git submodule= 添加到 [[https://github.com/ren-lingyu/resume-template/tree/main][main]] 分支. 

* 快速开始
1. =clone= 本项目 [[https://github.com/ren-lingyu/resume-template/tree/main][main]] 分支到本地. 从 [[https://github.com/ren-lingyu/resume-template/tree/fonts][fonts]] 分支下载所需字体. 
    #+begin_src bash
    git clone --depth 1 https://github.com/ren-lingyu/resume-template.git
    #+end_src
2. 进入简历目录.
    #+begin_src bash
    cd resume
    #+end_src
3. 完善简历后, 使用 latexmk 编译.
    #+begin_src bash
    latexmk -pdf main.tex
    #+end_src
